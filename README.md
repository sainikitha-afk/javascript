# JavaScript Fundamentals Tasks

This repository contains a collection of hands-on JavaScript projects completed as part of my internship training. The tasks focus on building interactive web applications using core JavaScript concepts, DOM manipulation, and event handling without relying on external frameworks.

---

## Overview

The goal of these tasks is to strengthen understanding of:
- JavaScript fundamentals
- DOM manipulation
- Event-driven programming
- Asynchronous operations
- UI interactivity and user experience

Each task is organized in a separate folder and includes:
- Source code (HTML, CSS, JavaScript)
- A detailed README explaining the implementation
- Screenshots or GIFs demonstrating functionality

---

## Tasks Implemented

### Task 1: To-Do List Application
- Add, edit, complete, and delete tasks
- Dynamic DOM updates and event handling

### Task 2: Simple Calculator
- Perform arithmetic operations
- Dynamic input handling and real-time display updates

### Task 3: Image Gallery Lightbox
- Clickable image thumbnails
- Modal view with dynamic content updates

### Task 5: Dynamic Quiz Application
- Multiple-choice quiz with score tracking
- Dynamic rendering of questions and results

### Task 6: Drag and Drop List
- Reorder list items using HTML5 Drag and Drop API
- Visual feedback for better user experience

### Task 7: Real-time Chat Simulation
- Simulated chat interface with timestamps
- Automated responses using timers

### Task 8: Single Page Application (SPA)
- Hash-based routing using JavaScript
- Dynamic content rendering without page reload

### Task 9: Infinite Scrolling Loader
- Load content dynamically on scroll
- Asynchronous data handling using Promises
